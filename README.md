# Sohib AI v5 Ultimate

Personal AI Companion berbasis web dengan fitur Living AI, Memory Management (Short/Long term), Emotion Detection, dan integrasi Voice. Dibangun sebagai Progressive Web App (PWA) modern.

**Created by Yostandi**

## Fitur Utama
- **Living AI:** Memiliki rutinitas dan kesadaran konteks.
- **Dynamic Identity:** Pengaturan nama, umur, gender, dan karakter saat pertama kali dijalankan.
- **Memory Engine:** Penyimpanan memori jangka pendek dan panjang untuk percakapan berlanjut.
- **Relationship System:** Level hubungan berkembang seiring interaksi.
- **Voice Integrations:** STT (Speech-to-Text), TTS (Text-to-Speech), dan Continuous Listening.
- **UI/UX:** Modern Glass UI, animasi Avatar, dan Full Offline PWA Cache.
- **AI Providers:** Mendukung OpenRouter & Ollama (Local LLM).

## Instalasi
1. Clone repositori ini.
2. Jalankan `npm install`.
3. Salin `.env.example` menjadi `.env` dan isi konfigurasi.
4. Jalankan `npm run dev` untuk development.
